### TortoiseGit 로그에서 파일 목록 가져오기
#### Get list of files from TortoiseGit log

    TortoiseGit > Show log

1. 로그 목록 에서 대상을 선택하고 로그를 복사한다 (full data copy)

### CMD Git 로그에서 파일 목록 가져오기

    명령 프롬프트(PowerShell) > 프로젝트 폴더 > git log --name-status
